Descripción del Curso

Este curso introduce los principios para diseñar y utilizar adecuadamente sistemas que incrementan su desempeño con base en la experiencia. El curso busca encontrar y reconocer patrones de comportamiento, ası́ como caracterı́sticas estructurales en conjuntos de datos de diverso origen para identificar elementos de manera automática, realizar predicciones, y aprender leyes de control. Se abordará los principales métodos utilizados para el aprendizaje supervisado, aprendizaje no supervisado, y aprendizaje por refuerzo (reinforcement learning), ası́ como una introducción a deep learning y a las redes neuronales convolucionales.
